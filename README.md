# Experiment 69: Unique Elements (HashSet)

## Problem Statement
Read $N$ integers from the user. Use a `HashSet` to remove duplicates and print only the unique numbers.

## Input Format
* **Line 1:** An integer $N$.
* **Line 2:** $N$ space-separated integers.

## Output Format
* Unique numbers (space separated).

## Example 1

**Input:**
```text
5
10 20 10 30 20
```


**Output:**
```text
10 20 30
```
